\doxysubsection{Descripción detallada}
Sistema embebido para el monitoreo y control ambiental en un invernadero de cannabis medicinal. 

Este programa gestiona un invernadero mediante sensores y actuadores que optimizan las condiciones de cultivo. Incluye captura de datos de temperatura, humedad relativa y luz ambiental, control de un LED RGB, y monitoreo remoto mediante puerto serie.
